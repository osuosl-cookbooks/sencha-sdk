Sencha SDK Cookbook
===================
This cookbook sets up a dev environment for the Sencha SDK. This environment
includes the Java, Ruby, and NPM environments, and the Sencha Touch and Sencha
CMD tools.

Requirements
------------

depends          'apt'
depends          'nodejs'
depends          'npm'
depends          'java'
depends          'ruby'
